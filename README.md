# Water Adsorption on Silica using LAMMPS

## Overview
This project investigates the adsorption of water molecules on amorphous and crystalline silica surfaces using Molecular Dynamics (LAMMPS).

## Objectives
- Study water adsorption behaviour on silica surfaces.
- Compare amorphous and crystalline silica.
- Analyse molecular interactions and adsorption trends.

## Tools Used
- LAMMPS
- Molecular Dynamics (MD)
- Grand Canonical Monte Carlo (GCMC)
- OVITO

## Results
- Successfully simulated water adsorption.
- Analysed adsorption behaviour and molecular interactions.
- Generated simulation outputs for comparison.
